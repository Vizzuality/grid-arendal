The Global Environment Outlook GEO 3 Electronic References

The GEO 3 Electronic Reference Scheme is a system developed to facilitate easy access to cited resources through CD-ROM and on the Web.

It allows for searches on references to be performed in three categories:

  1. Search by Author:
    A query based on the author's name(s) as they appear in the GEO 3 publication.
  2. Search by Title:
    A query based on the title of cited resource(s)as they appear in the GEO 3 publication.
  3. Search By GEO Tag:
    A mechanism that uniquely identifies each electronic reference as realized in the GEO 3 publication.
