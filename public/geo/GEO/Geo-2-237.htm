General information

The system provides users with access to Fishery Statistics of various sorts. Any data having time series structure can potentially be stored and processed by Fishstat Plus. The system consists of the main module and the datasets. Each dataset can be installed and uninstalled separately. See the list of available datasets below.

Key features

  1. Windows-based interface with menus and toolbar. Context-dependent popup menus are available by clicking right mouse button.
  2. Easy to install
  3. Filtering capabilities for selecting only data you want
  4. Aggregation and grouping by many group types.
  5. Simply start typing to find an object (country, species, etc.)
  6. Formula builder allows standard statistical functions as well as user-defined formulas
  7. Fast and simple data transfer to other applications through clipboard
  8. Advanced printing and exporting capabilities, including cross-tables and charts, using Report Wizard.
  9. Portability (any version of Windows)
  10. Context Help system.

System Requirements:

  1. Windows 3.1, Windows 95, Windows 98 or Windows NT
  2. 10 MB of hard disk for the system. Additional space for the datasets (2 - 12 Mb per dataset)
  3. 8 MB RAM with Windows 3.1, 16 with Windows 95 or NT
  4. Colour VGA monitor, SVGA recommended

How to download and install the software

  1. Download Fishstat Plus:
    Click here to start download (Version 2.3) Ftp.fao.org/fi/stat/windows/fishplus/fst_plus.zip (6.1 Mb)
  2. Uncompress files into any directory (preferably empty) using PKUNZIP or WINZIP
  3. Start SETUP.EXE
  4. Confirm installation directory C:\FISHSTAT, or change directory
  5. A new Program Manager group or Start menu item called "Fishstat Plus" will be created.
  6. Go on to the following paragraph to install one or more datasets.

How to download and install the datasets

  1. Download the dataset(s) that you are interested in into any directory. If you have Fishstat Plus version 2.30 or newer (recommended), there is no need to decompress the datasets.
    • Aquaculture production: quantities 1970-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/aquaq.zip (0.8 Mb)
      Data included in the dataset are published as:
      FAO yearbook, Fishery statistics, Aquaculture production 1999. Vol 88/2. Formerly published as: FAO Fisheries Circular N.815 Rev.11 "Aquaculture Production Statistics"
    • Aquaculture production: values 1984-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/aquav.zip (0.8 Mb)
      Data included in the dataset are published as:
      FAO yearbook. Fishery statistics. Aquaculture production 1999. Vol. 88/2. Formerly published as: FAO Fisheries Circular N.815 Rev. 11 "Aquaculture Production Statistics"
    • Capture production 1970-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/capdet.zip (1.4 Mb)
      Data included in the dataset are published as :
      FAO yearbook. Fishery statistics. Capture production 1999, Vol. 88/1.
    • Total production 1950-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/fstat.zip (1.6 Mb)
      Data up to 1995 included in the:
      FAO Yearbook of Fishery Statistics "Catches and Landings", Vol.80
      This Yearbook is no longer published
    • Commodities trade and production of fishery products 1976-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/fishcomm.zip (2.6 Mb)
      Please click here to read important note.
      Data included in the dataset refer to:
      FAO Yearbook, Fishery Statistics, Commodities 1999, Vol.89
    • Eastern Central Atlantic capture production 1970-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/cecaf.zip (0.6 Mb)
      Data up to 1991 included in the:
      FAO Bulletin of Fishery Statistics "CECAF Nominal Catches", Vol.7
      This Bulletin is no longer published
    • Mediterranean and Black Sea capture production 1970-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/gfcm.zip (0.6 Mb)
      Data up to 1991 included in the:
      FAO Bulletin of Fishery Statistics "GFCM Nominal Catches", Vol.9
      This Bulletin is no longer published
    • Southeast Atlantic capture production 1975-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/atl_se.zip.zip (0.6 Mb)

    Datasets produced by other agencies:

    • Indian Ocean Tuna Commission Nominal Catch database:
      Download Ftp.fao.org/fi/stat/windows/fishplus/iotc.zip (0.2 Mb)
      Produced by Indian Ocean Tuna Commission
    • Atlantic capture 1970-1998:
      Download Ftp.fao.org/fi/stat/windows/fishplus/atlant.zip (1.7 Mb)
      Produced by CWP
    • Las Palmas Survey (LPS) 1991-1999:
      Download Ftp.fao.org/fi/stat/windows/fishplus/lps.zip (0.1 Mb)
      Data provided by the "Centro Oceanografico de Canarias, Instituto Español de Oceanografía"
  2. Start Fishstat Plus
  3. Invoke File-Install datasets command
  4. In Install dialog, change directory to where you have downloaded dataset(s).
  5. When the text on top of the window shows directory path where dataset files are located, the dataset names will appear in the list on the right. Click on a dataset name, then on Install button
  6. Repeat steps 3-5 for all datasets that you have downloaded
