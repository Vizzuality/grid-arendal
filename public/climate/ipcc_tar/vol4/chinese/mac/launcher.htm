Implementation Comments and Customization Tips:

  • Name "qviewer" have been assigned to this frame in order to be able to open found documents in it. (This was an optional step used for the purpose of demonstration only.)
  • All means of starting search applet provided above invoke JavaScript function startQuestAgent that opens a new window with search applet in it.
  • To change appearance of any search applet component or to change any message (to translate applet to other language for example) review qagent.prm file located in directory j:\ipcc\test\ca03e .
  • You can change applet size using WIDTH and HEIGHT applet parameters defined in this file but you might need to modify browser window size defined in this file too.
