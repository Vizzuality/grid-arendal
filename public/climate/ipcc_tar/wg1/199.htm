5.4.3 Modelling the Indirect Effect of Aerosols on Global Climate Forcing

Published estimates of the indirect forcing by anthropogenic aerosols are summarised in Table 5.11. The results in Table 5.11a are for the assumption that changes in initial droplet concentrations do not lead to any changes in cloud properties. These results, therefore, did not include any feedbacks to liquid-water content or cloud amount as a result of changes to precipitation efficiency. The results in Table 5.11b include a calculation of the decrease in precipitation efficiency that results from an increase in Nd. Therefore, these results do include feedbacks to changes in liquid-water and cloud amount from changes in cloud microphysics.
